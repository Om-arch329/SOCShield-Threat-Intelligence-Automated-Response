# SOCShield

SOCShield – Intelligent Web Attack Detection, Threat Intelligence, and Automated Response Platform.

A Security Operations Center (SOC) platform for web attack detection, threat intelligence enrichment, automated response, and security monitoring using Wazuh, ModSecurity, VirusTotal, and MITRE ATT&CK.

## Technologies Used

- Wazuh SIEM
- ModSecurity WAF
- NGINX
- DVWA
- VirusTotal
- MITRE ATT&CK
- Postfix
- Linux

## Features

- Web Attack Detection (XSS, SQL Injection, RCE, Command Injection)
- Threat Intelligence Enrichment
- MITRE ATT&CK Mapping
- Automated Active Response
- Email Alerting
- Custom Security Dashboards
- Threat Hunting

## Security Notice

All API keys, credentials, email addresses, and sensitive configuration values have been removed or redacted prior to publication.
